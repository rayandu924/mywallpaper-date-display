# Date Display Widget for MyWallpaper

A beautiful and customizable date display widget for [MyWallpaper](https://mywallpaper.app).

![Date Display Preview](preview.png)

## Features

- **Multi-language support** - Choose from 11 preset languages or define your own custom day/month names
- **Customizable fonts** - Select from popular Google Fonts or provide your own font URL
- **Flexible date formats** - Multiple format options including long, short, numeric, ISO, and more
- **Full style control** - Font size, weight, color, opacity, letter spacing
- **Text effects** - Configurable text shadow with color, blur, and offset controls
- **Real-time updates** - Automatically updates at midnight

## Supported Languages

- English, Français, Deutsch, Español, Italiano, Português
- 日本語, 中文, 한국어, Русский, العربية

## Date Formats

| Format | Example |
|--------|---------|
| Long | January 20, 2026 |
| Short | Jan 20, 2026 |
| Numeric | 20/01/2026 |
| Numeric US | 01/20/2026 |
| ISO | 2026-01-20 |
| Day & Month | 20 January |
| Month & Day | January 20 |

## Settings

### Display Options
- Show/hide day of week
- Show/hide date
- Choose date format

### Language
- **Preset Mode**: Select from built-in languages
- **Custom Mode**: Define your own day and month names (comma-separated)

### Font
- **Preset Fonts**: Inter, Roboto, Outfit, Poppins, Montserrat, Open Sans, Lato, Playfair Display, Merriweather, Source Code Pro, JetBrains Mono
- **Custom URL**: Provide a Google Fonts URL or direct font file URL

### Style
- Font sizes for day and date
- Font weight (300-800)
- Text color
- Text alignment (left, center, right)
- Opacity
- Letter spacing

### Effects
- Text shadow with customizable:
  - Shadow color
  - Blur amount
  - X/Y offset

## Installation

1. Open MyWallpaper
2. Go to the Addons section
3. Search for "Date Display"
4. Click Install
